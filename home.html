Welcome to Crowdfunding Data Analytics

This web application presents crowdfunding campaign analysis in a simple, interactive, and visually attractive way. It helps users understand project distribution, funding performance, and campaign success using Tableau dashboards and insights.

View Dashboards View Insights Give Feedback

Project Objective

To analyze crowdfunding data and identify patterns related to category popularity, funding performance, donor support, and campaign success.

Tools and Technologies Used

Excel, Python, Tableau, HTML, CSS, JavaScript, Node.js, and Express.js

What Users Can Explore

Interactive dashboards, key findings, alert-based actionable insights, and the Tableau story.

Why This Application is Useful

This application helps entrepreneurs, researchers, and analysts understand crowdfunding trends more clearly. Users can explore category-wise project distribution, funding gaps, backer behavior, and the factors influencing campaign success.
